## Grade & Simple Calculator

A beginner-friendly web project built using HTML, CSS, and JavaScript.
This project contains:

* Grade Calculator

* Simple Calculator

It is fully responsive with basic animations and interactive buttons.

## Features
* Grade Calculator

* Takes user marks as input

* Displays grade based on score

* Shows emoji feedback 😄🤩

* Animated card design

* Simple Calculator

* Takes two number inputs

## Performs:

➕ Addition

➖ Subtraction

✖ Multiplication

➗ Division

Displays result instantly

## Technologies Used

* HTML5

* CSS3 (Animations, Flexbox, Shadows)

* JavaScript (DOM Manipulation)
